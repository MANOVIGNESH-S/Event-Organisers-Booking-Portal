# 🎉 Event Organisers Booking Portal

A full-stack event management platform where organizers can create and manage events, and users can register, book slots, and handle payments — all through a responsive web interface.

> **Stack:** React.js · Spring Boot · MySQL · REST API  
> **Duration:** Jan 2024 – Feb 2024

## ✨ Features

- 🔐 User registration & JWT-based authentication
- 📅 Event creation, listing & management for organizers
- 🎟️ End-to-end booking workflow — browse, select, confirm
- 💳 Payment integration via REST APIs
- 📱 Responsive UI built with React.js

---

## 🗂️ Project Structure

Event-Organisers-Booking-Portal/
├── backend/     # Spring Boot REST API (Java + Maven)
└── frontend/    # React.js frontend (Create React App)

---

## 🚀 Getting Started

### Prerequisites
- Node.js & npm
- Java 17+
- Maven
- MySQL

### Backend

```bash
cd backend
# Set your DB credentials in src/main/resources/application.properties
./mvnw spring-boot:run
```
Runs at `http://localhost:8080`

### Frontend

```bash
cd frontend
npm install
npm start
```
Runs at `http://localhost:3000`
